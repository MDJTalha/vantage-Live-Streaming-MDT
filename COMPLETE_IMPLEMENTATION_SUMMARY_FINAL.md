## 📋 QUICK REFERENCE

### Startup Commands

```bash
# Option 1: Automated (Recommended)
# Double-click: START-ALL.bat

# Option 2: Manual
# Terminal 1: API
cd apps\api && npm run dev

# Terminal 2: Web
cd apps\web && npm run dev

# Terminal 3: Media
cd apps\media-server && npm run dev
```

### Database Setup

```bash
# Using Rancher Desktop
cd c:\Projects\Live-Streaming-
"C:\Program Files\Rancher Desktop\resources\resources\win32\bin\docker-compose.exe" -f docker-compose.databases.yml up -d
```
